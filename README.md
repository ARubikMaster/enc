# enc
v.1.0.0
<hr>

### About
Enc is a basic file encrypter.
It is literally just a wrapper for the built in crypto functions lol

### How to use
```
enc encrypt <filename>
enc decrypt <filename.enc>
```

### How files are stored
```
<filename>\n<encrypted content>\n
```
